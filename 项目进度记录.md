## 二、技术架构

### 前端
- Vue 3 + Vite 5 + TypeScript
- Tailwind CSS（样式框架）
- Pinia（状态管理）
- Vue Router（路由）
- Axios（HTTP 请求）
- Heroicons（图标库）

### 后端
- Express 4
- JWT 身份认证
- bcryptjs（密码加密）
- WebSocket（ws 库）
- JSON 文件存储（`data/` 目录）

### 三种角色
1. **学生用户** - 前端首页直接访问，无需登录，下单时填手机号即可
2. **超级管理员** - `/super-admin/login`，手机号免密登录，默认账号：`13800138000`
3. **代理商/业务员** - `/agent/login`，用户名免密登录，默认账号：`agent001`

### 超级管理员特权列表
超级管理员拥有平台最高权限，具体包括：

**1. 订单管理**
- 查看所有订单（全平台）
- 查看订单详情（含用户信息、文件、转账截图）
- 确认收款（pending_confirm → paid）
- 取消订单
- 分配订单给业务员
- 从业务员回收订单
- 删除订单

**2. 业务员管理**
- 查看所有业务员列表
- 创建新业务员（账号、密码、姓名、手机号、负责业务类型）
- 修改业务员信息（姓名、手机、服务类型、状态、重置密码）
- 启用/禁用业务员
- 删除业务员
- 查看业务员收入统计

**3. 用户管理**
- 查看所有注册用户列表
- 查看用户详情（订单数、消费总额）
- 启用/禁用用户账号

**4. 服务定价管理**
- 修改所有服务单价（打印、快递、课业、写字等）
- 修改最低提现金额
- 所有价格修改实时生效

**5. 财务管理**
- 查看交易流水（所有支付记录）
- 对账汇总（按日期查询收支统计）
- 审核业务员提现申请（通过/驳回）
- 上传/更换收款码（支付宝、微信）

**6. 运营管理**
- 查看用户评价
- 查看操作日志（所有管理员的操作记录）
- 数据统计看板（今日/本周/本月数据）

### Token 存储
- 学生用户：`localStorage.token`
- 超级管理员：`localStorage.super_admin_token`
- 代理商：`localStorage.agent_token`

## 三、功能模块

### 3.1 学生前端

#### 首页 (Home.vue)
- Hero 区域 + 服务分类卡片
- 公告栏跑马灯（CSS 动画，25秒/圈）
- 4 个付费服务卡片：文件打印、课业时长整理、校内包裹配送、写字服务
- 4 个免费工具卡片：PDF转换、证件照制作、图片压缩、图片格式转换
- 联系客服弹窗（微信号：wxid_jv428py2ftir22）
- 移动端底部导航（首页/订单/客服/我的）

#### 订单系统
- 4 种服务类型：打印、课业时长整理、快递配送、写字
- 下单时只需填手机号，无需注册登录
- 防重复下单：5分钟内同手机号同类型待支付订单拦截
- 订单超时取消：30分钟未支付自动取消（后端定时任务，每2分钟检查一次）
- 订单查询：手机号即可查询，无需登录
- 订单状态：待支付(pending)、待确认收款(pending_confirm)、已支付(paid)、处理中(processing)、已完成(completed)、已取消(cancelled)

#### 支付系统
- 人工转账确认模式（无支付宝/微信商户号）
- 用户上传转账截图 → 管理员后台确认收款
- 支付页 4 步引导流程

#### 用户评价系统
- 已完成订单可评价（5星评分 + 文字评价）
- 管理员后台可查看所有评价统计

#### 免费工具（纯前端，不上传服务器）
1. **PDF转换** (PdfConvert.vue) - 6种转换：图片转PDF、PDF转图片、文本转PDF、PDF转文本、Word转PDF、合并PDF
2. **证件照制作** (IdPhoto.vue) - 智能抠图（四角颜色检测+边缘平滑）、手动选色、多种背景色
3. **图片压缩** (ImageCompress.vue) - 批量压缩、质量滑块、大小对比
4. **图片格式转换** (ImageConvert.vue) - JPG/PNG/WebP 互转

#### 个人中心 (Profile.vue)
- 手机号查询订单
- 我的订单 / 余额钱包 / 收货地址 菜单
- 联系客服入口
- 退出登录

### 3.2 超级管理员后台 (/super-admin)

#### 功能菜单
- 首页概览（统计卡片）
- 订单管理（全部订单、待确认收款、订单池）
- 代理商管理（增删改查）
- 用户管理
- 服务定价配置
- 财务管理（平台收入、代理商提现审核、对账汇总、支付流水）
- 操作日志
- 用户评价

#### 对账汇总
- 按日期范围统计订单数、收入、平台收入、代理商分成
- 每日明细表格

#### 操作日志
- 记录所有管理员关键操作（确认收款、分配订单、审核提现等）
- 支持分页查看

### 3.3 代理商后台 (/agent)

- 订单池（抢单）
- 我的订单
- 我的钱包（余额、提现、明细）
- 个人资料

## 七、部署相关

### 本地开发
```bash
# 前端
npm run dev      # 启动开发服务器
npm run build    # 构建生产版本

# 后端
node api/server.js   # 启动后端 (或用 --watch 自动重载)
```

### 后端端口
- 后端默认端口：3000（或环境变量 PORT）
- 前端开发通过 vite.config.ts 的代理转发 `/api` 到后端
